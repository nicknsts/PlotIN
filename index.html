FAQ

Which OS are supported?

Windows (EXE). The codebase is Python with Tkinter + Matplotlib.

Can I export EMF?

Yes. EMF is produced via SVG conversion. Native SVG/PDF/PNG are included.

Math/LaTeX‑style labels?

Supported: Greek letters, subscripts, exponents, scientific symbols.

Demo vs Pro?

Demo is for quick evaluation; Pro unlocks all exports and regular updates.
