Key features

Generate plots instantly by paste, then adjust only what you need: legend, axes, fonts and export format.

Paste-based workflow

Paste tabular data (e.g. from Excel/CSV). PlotIN auto-detects headers and builds X–Y series for you.

Legend & series names

Drag the legend, change its size, rename series inline, and show/hide the legend without digging through menus.

Math-style labels

Greek letters, subscripts, superscripts and full mathtext — from simple units (Ω, °C, μ) to advanced expressions such as Vₒᵤₜ(t) = 5·e⁻ʲωt, Z(ω) = R + jωL, or H(s) = ω₀² / (s² + 2ζω₀s + ω₀²).

Axes & font control

Use Figure options (General & Axes tabs) to adjust axis limits, label fonts, tick density and hide or show axes/ticks.

Zoom & navigation toolbar

Zoom horizontally or vertically, pan, and use Back/Forward, Home (short & long press) to navigate views safely.

Export & portability

Export plots as SVG, PDF, PNG, EMF (via SVG conversion) & others. PlotIN is shipped as a portable executable on Windows and macOS: no installation required, works fully offline.

FAQ

Math / LaTeX-style labels?

Yes. PlotIN supports Greek letters (\alpha, \Omega), subscripts (V_{out}), exponents (x^2) and common scientific symbols. Labels are rendered using Matplotlib’s mathtext engine. See the label syntax guide for examples.

Which OS are supported?

PlotIN is available as a portable .exe on Windows and as a drag-and-drop .app bundle on macOS. Both require no installation.

Can I export EMF?

Yes. EMF is generated from the SVG using Inkscape’s command-line tools. You need Inkscape installed and available in your system PATH. If Inkscape is missing, EMF export is disabled, but SVG/PDF/PNG remain available.

Do I need to install anything?

No installer is required. On Windows, run the .exe directly. On macOS, drag the .app anywhere you like (Desktop, Applications, USB drive) and launch it. PlotIN works fully offline; internet is only needed to download updates.

Is PlotIN free?

The current Student version is free for personal and academic use. A separate Pro edition with advanced features (for example multi-format export in a single step or per-series style control) may be offered in the future.
