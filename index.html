FAQ

Math / LaTeX-style labels?

Yes. PlotIN supports Greek letters (\alpha, \Omega), subscripts (V_{out}), exponents (x^2) and common scientific symbols. Labels are rendered using Matplotlib’s mathtext engine. See the label syntax guide for examples.

Which OS are supported?

PlotIN is available as a portable .exe on Windows and as a drag-and-drop .app bundle on macOS. Both require no installation.

Can I export EMF?

Yes. EMF is generated from the SVG using Inkscape’s command-line tools. You need Inkscape installed and available in your system PATH. If Inkscape is missing, EMF export is disabled, but SVG/PDF/PNG remain available.

Do I need to install anything?

No installer is required. On Windows, run the .exe directly. On macOS, drag the .app anywhere you like (Desktop, Applications, USB drive) and launch it. PlotIN works fully offline; internet is only needed to check for the updates and download new versions.

Is PlotIN free?

The current Student version is free for personal and academic use and will remain free. A separate Pro edition with additional features may be offered later.

Does PlotIN collect any data or telemetry?

No. PlotIN does not collect usage data, telemetry, or personal information. All processing happens locally. The app only performs an optional request to check if a new version is available.
