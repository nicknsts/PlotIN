Key features

Start with an instant, fully automatic plot — then adjust only if you want to. Axes, labels, legend, zoom and export are all under your control, and everything updates live.

Paste-based workflow

Paste tabular data from Excel, CSV or any spreadsheet. PlotIN auto-detects headers or numeric rows, builds X–Y series automatically, and handles unlimited rows and columns. Once pasted, the data remains fully editable — with instant preview updates and complete undo/redo support for quick, safe adjustments.

Legend & series names

Drag and resize the legend, rename series inline, and toggle visibility with a single click. The legend uses a minimalist, semi-transparent scientific style with full MathText support (Matplotlib’s formula engine), and its content scales automatically as you resize it. Updates from other menus apply instantly without shifting its position or size.

Math-style labels

Greek letters, subscripts, superscripts and full MathText expressions — from simple units (Ω, °C, μ) to advanced notation such as ∑ₖaₖe⁻ʲωₖt, ∫x(t)dt, ∂V/∂x or ∇·E = ρ/ε₀. Axis titles accept MathText directly from a single menu, letting you enter complex scientific structures that PlotIN formats instantly — capabilities that many commercial tools either lack entirely or make difficult to achieve.

Axes & font control

Gain full control over all four axes. Each axis can be shown or hidden, while major and minor ticks can be tuned precisely. Major ticks are computed intelligently to avoid clutter and maintain readable scales. Axis limits can be set manually, and font sizes for titles and tick labels can be customized easily for publication-quality output.

Zoom & navigation toolbar

Zoom in, out, horizontally or vertically, and pan freely in any direction for precise manual control. Navigate between views using Back and Forward, and rely on the Home button for safe resets: a short press restores the default framing, while a long press performs a full reset and returns to the original view.

Export & portability

Export plots as SVG, PDF, PNG, EMF (via SVG conversion) and more. Vector formats keep infinite sharpness, while raster exports use high-resolution settings for clean results. The exported figure always matches the on-screen view — aspect ratio, zoom level, legend placement and all adjustments are preserved exactly. PlotIN ships as a portable executable for Windows and macOS, with no installation needed and full offline functionality.

FAQ

Math-style labels?

Yes. PlotIN supports Greek letters (\alpha, \Omega), subscripts (V_{out}), exponents (x^2) and a wide range of scientific symbols. Labels are rendered using Matplotlib’s MathText engine. See the label syntax guide for examples and advanced formatting.

Which OS are supported?

PlotIN is available as a portable .exe on Windows and as a drag-and-drop .app on macOS. No installation is required on either platform.

Can I export EMF?

EMF export uses Inkscape’s command-line tools. If Inkscape is not installed or not found in your PATH, EMF is disabled automatically — but SVG, PDF and PNG remain available.

Do I need to install anything?

No installation is required. On Windows, run the .exe directly. On macOS, drag the .app anywhere you like and launch it. PlotIN works fully offline; internet is only needed to check for updates or download new releases.

Is PlotIN free?

The current Student version is free for personal and academic use and will remain free. A separate Pro edition with additional features may be introduced later.

Does PlotIN collect any data or telemetry?

PlotIN uses a minimal, privacy-safe telemetry system. It does not collect personal information of any kind. The app records only a few anonymous technical metrics — operating system, number of launches, and approximate unique-user count — to help guide development and improve stability. Telemetry is strictly anonymous and cannot identify users.
